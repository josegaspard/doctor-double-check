Información del proyecto

Usar josegaspard.dev
Los cambios realizados mediante josegaspard.dev se guardarán automáticamente en este repositorio.

Este proyecto está construido con:

Vite
TypeScript
React
shadcn-ui
Tailwind CSS

¿Cómo puedo desplegar este proyecto?

Simplemente abre josegaspard.dev
 y haz clic en Share → Publish.

¿Puedo conectar un dominio personalizado a mi proyecto en josegaspard.dev?

Sí, ¡puedes hacerlo!

Para conectar un dominio, ve a Project → Settings → Domains y haz clic en Connect Domain.
