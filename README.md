¿Cómo puedo editar este código?

Existen varias formas de editar tu aplicación.

Usar josegaspard.dev

Simplemente visita el Proyecto en josegaspard.dev
 y comienza a escribir prompts.

Los cambios realizados mediante josegaspard.dev se guardarán automáticamente en este repositorio.

Usar tu IDE preferido

Si deseas trabajar de forma local usando tu propio IDE, puedes clonar este repositorio y subir los cambios.
Los cambios enviados también se reflejarán en josegaspard.dev.

El único requisito es tener Node.js y npm instalados — instalar con nvm

Sigue estos pasos:

# Paso 1: Clona el repositorio usando la URL Git del proyecto.
git clone <YOUR_GIT_URL>

# Paso 2: Navega al directorio del proyecto.
cd <YOUR_PROJECT_NAME>

# Paso 3: Instala las dependencias necesarias.
npm i

# Paso 4: Inicia el servidor de desarrollo con recarga automática y vista previa instantánea.
npm run dev


Editar un archivo directamente en GitHub

Navega al archivo o archivos deseados.

Haz clic en el botón "Edit" (icono de lápiz) en la parte superior derecha de la vista del archivo.

Realiza los cambios y confírmalos (commit).

Usar GitHub Codespaces

Ve a la página principal de tu repositorio.

Haz clic en el botón "Code" (botón verde) cerca de la parte superior derecha.

Selecciona la pestaña "Codespaces".

Haz clic en "New codespace" para iniciar un nuevo entorno Codespace.

Edita los archivos directamente dentro de Codespaces y realiza commit y push cuando termines.
